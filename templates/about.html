About the Subnetting App

Hi,

I am Alap. Welcome to my subnetting App. When I was studying for the CCNA, I had been looking for a website that would help me practice subnetting and also time me for each question. I couldn't find anything, so I wrote a python script that I would use to practice. The script would generate subnetting questions for me, time me for each question, and provide the average time taken. After I passed the CCNA, I decided to convert that script into this web app. I thought it might be of use to other aspirants as well. I chose the Flask framework as it suited well to my requirements. While working on the app, I realized this was a great opportunity to study and integrate certain DevOps concepts. That turned this project into a full-on DevOps project, as I used Jenkins and Docker to build, test, and deploy the app. You can read more about it at https://github.com/alapvyas/Subnetting_project.git

Please reach out to me at example@email.com if you have any questions. Feel free to reuse my script and source code for this app.
